# QR Codes for Virasat – Namma Guide

Each QR code encodes the **Site ID** as raw text. Use any QR generator
(e.g. qr-code-generator.com) to create printable codes for demo.

| Site ID    | Site Name                      |
|------------|-------------------------------|
| SITE_001   | Hoysaleswara Temple           |
| SITE_002   | Chennakeshava Temple          |
| SITE_003   | Hire Benakal Megalithic Site  |
| SITE_004   | Aihole Rock Inscriptions      |
| SITE_005   | Shivagange Step Wells         |
| SITE_006   | Badami Cave Temples           |

## How to generate a QR code:
1. Go to https://www.qr-code-generator.com/
2. Choose **Text** type
3. Enter exactly: `SITE_001` (or any Site ID above)
4. Download and print

Scanning `SITE_001` in the app will open Hoysaleswara Temple's detail
page and allow unlocking the Hidden Fact.
